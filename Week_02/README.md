## 深度优先遍历 和 广度优先遍历
场景：树的遍历 和 图的遍历

深度优先 使用 栈的数据结构 （比如递归实现）

广度优先 使用 队列的数据结构

---

举一个形象点的例子说明**深度优先**和**广度优先**的区别

**你去挖黄金**

1.你沿着一个地点一直挖下去，直到挖不动为止。然后再换一个地方挖（这是深度优先）
2.一个地点一次你只挖一锄，然后换一个地方挖，直到所有地方都挖过了后再向深处挖（这是广度优先）


### 对算法学习的总结
1. 一时搞不懂原理，自己想不出来思路 别死扣，这样挺浪费时间的。  可以网上看看别人对该算法的一个具体的分析，思路。 从结果再来倒推过程
最终的目的都是学习，理解，掌握该算法。  按成本最低的方式去掌握（时间成本）。 我们遇到的大多数问题都不是一个全新的问题，很可能其他人已经遇到过，也存在了相应的解。
